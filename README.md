# Nexium Webhook

![Manual Discord Test](https://github.com/buildsbybuchanan/nexium-webhook/actions/workflows/manual-discord-test.yml/badge.svg)

Nexium Webhook is a public automation showcase by BuildsByBuchanan.

It demonstrates:
- GitHub Actions
- Discord webhook notifications
- Secure GitHub Actions secrets
- Repository automation workflows

No production secrets are stored in this repository.
